# nvim-config

A minimal Neovim configuration using `vim.pack`

> **Requires Neovim 0.12+** â€” currently nightly on Windows.

## Features

- LSP via `nvim-lspconfig` + `mason`
- Autocompletion via `blink.cmp`
- Snippets via `LuaSnip` + `friendly-snippets`
- Fuzzy finding via `telescope.nvim`

## Install

**Windows (PowerShell)**
```powershell
git clone https://github.com/davidheeren/nvim-config "$env:LOCALAPPDATA\nvim"
```

**Windows (CMD)**
```cmd
git clone https://github.com/davidheeren/nvim-config %LOCALAPPDATA%\nvim
```

**Linux**
```bash
git clone https://github.com/davidheeren/nvim-config ~/.config/nvim
```
